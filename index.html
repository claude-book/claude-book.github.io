Os códigos do livro Do Chat ao Cowork e Code

Este é o lar dos projetos criados ao longo do livro de Henrique Alvarenga sobre o Claude Code, o agente de programação da Anthropic. Cada repositório aqui é um exemplo citado nos capítulos — você pode clonar, rodar, estudar e adaptar.

Sobre o livro

Escrito para quem quer construir software usando vibe coding — conversando em linguagem natural para a IA escrever o código. Do primeiro chat até projetos completos, o livro mostra, com casos reais, como transformar ideias em ferramentas que funcionam — como um servidor MCP que cria e publica Google Forms.

Repositórios

mcp-server-google-forms Servidor MCP que dá ao Claude o poder de criar, editar e publicar Google Forms. Vem com site próprio, documentação e a revisão de código contada no livro.
